01

What We
Measure

For every brand, we measure five things across AI: how often you're mentioned, where you rank against competitors, what AI gets factually wrong, which sources it pulls from, and whether your language matches what buyers actually ask. Each dimension requires hundreds of queries — because AI gives different answers every time.

Real User Interface Simulation

Most AI visibility tools query AI models through developer APIs. The problem: API responses often differ from what real users see in the chat interface. Different model versions, different context windows, different response formatting. We simulate real user sessions — what an actual person sees when they type a prompt into ChatGPT, Perplexity, or Gemini. This means the visibility data in your report reflects your customers' actual experience, not a developer endpoint approximation.

02

Why It's
Hard

AI answers are nondeterministic — the same question produces different recommendations each time. Some answers come from real-time web search, others from the model's training data. Each platform weighs different sources differently. A brand can be visible on one AI and invisible on another. That's why we query AI hundreds of times per report — to separate signal from noise.
